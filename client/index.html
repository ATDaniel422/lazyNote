About lazyNote

lazyNote is a web application designed to make note-taking easier. All you have to do is hit record and it starts sending your audio to our system. Once you are finished recording your audio, hit submit and lazyNote begins turning it into text.

This system is utilizes Amazon Web Services serverless architecture and AWS Transcribe speech-to-text API. Each serverless Lambda Function is built using Python, and coordinated using and AWS Step Function.
